# Skill — escrita-executiva-pt-br

> Skill para produzir textos em tom executivo brasileiro maduro.
> Versão 1.0 — 2026-07

---

## Para quem é

Para C-levels, heads de área, líderes que escrevem para conselho, diretoria, comunicados públicos, posts de autoridade no LinkedIn.

## Como instalar

### Em Claude Code

```bash
ln -s /caminho/para/deep-claude/skills/escrita-executiva-pt-br ~/.claude/skills/escrita-executiva-pt-br
```

### Em Cowork

Settings > Capabilities > Skills > Install from folder.

## Como usar

Após instalar, basta pedir:

> "Escreve uma nota de 400 palavras para o conselho sobre o trimestre"
> "Prepara um post de autoridade sobre IA generativa para LinkedIn"
> "Redige um comunicado executivo sobre a parceria com [empresa]"

## Diferenciador

A maioria das ferramentas produz texto que soa como IA: frases curtas, listas em tudo, clichês corporativos. Este skill produz texto com 6 características que distinguem voz humana sênior de voz de IA:

1. Frases longas com vírgulas
2. Sem cadência de IA
3. Conjunções ricas
4. Sem clichês corporativos
5. Densidade alta
6. Travessão raro

## Exemplos

**Antes do skill** (típico output de IA):

> Em primeiro lugar, a IA é importante. Ela traz benefícios. Por exemplo, automação, eficiência e inovação. Em segundo lugar, é fundamental adotar estratégias claras. Por fim, devemos olhar holisticamente para o futuro.

**Depois do skill**:

> A IA deixou de ser tema de inovação para virar infraestrutura operacional, e quem ainda trata o assunto como projeto piloto perde terreno todo trimestre. O que precisamos discutir nesta reunião não é se vamos adotar, mas com que cadência, em quais frentes, e sob qual governança, porque adoção sem governança vira passivo, e governança sem adoção vira teatro.
